#PENSAMIENTO COMPUTACIONAL

*  ABSTRACCION

¿Qué se solicita finalmente?

Un programa que válide el acceso a una plataforma virtual.

¿Qué información es relevante dado el problema anterior?

- Nombre y edad del Usuario
- Un indicador de credenciales válidas
- Contador del total de veces que el ususario intento acceder sin éxito

*   DESCOMPOSICION

¿Qué acciones se requieren para resolverlo?

- Ingresar nombre
- Ingresar edad
- Ingresar credenciales
- Validar credenciales
- Validar edad
- Registar numero de intentos fallidos
- Generar Mensaje 
- Retornar mensaje

*   RECONOCIMIENTO DE PATRONES

¿Qué puede reutilizar de la solución de otros problemas?

- ingresar_texto
- ingresar_entero
- mostrar_error
- mostrar_mensaje
